 En Debian GNU/Linux 4.0 "Etch"

 Para poder instalar el mysql que nos dieron para TALLER 3:

 1º Desinstalar los paquetes:
	- mysql-client
	- mysql-client-5.0

	NOTA: El paquete llamado: mysql-common no lo he podido desinstalar por motivos
	de problemas de dependencias.

 2.1º Obtener privilegios de root:
	shell:> su

 2.2º Agregar el grupo "mysql":  
	shell> groupadd mysql

 2.3º Agregar el usuario "mysql" perteneciente al grupo "mysql"
	shell> useradd -g mysql mysql

 2.4º Dejar los privilegios de root:
	shell:> exit

 3º Ubicarse en el directorio donde se decomprimirá la distribución binaria de mysql
   En mi caso:
	shell> cd /data/

 4º Descomprimir el paquete binario comprimido mediante:
	shell:> tar zxvf mysql-standard-5.0.24a-linux-i686.tar.gz

   Se creará el directorio mysql-standard-5.0.24a-linux-i686 con todo el contenido.
  
 5º Hacer un enlace simbólico al directorio para facilidad de uso, en mi caso:
	shell> ln -s /data/mysql-standard-5.0.24a-linux-i686 mysql

 6º Ubicarse en directorio mediante:
	shell> cd mysql

 7.1º Obtener privilegios de root:
        shell:> su

 7.2º Cambiar el usuario propietario del directorio al usuario "mysql":
	shell> chown -R mysql .

 7.3º Cambiar el grupo propietario del directorio al usuario "mysql":
	shell> chgrp -R mysql .

 7.4º Establecer una contraseña para el usuario "mysql", en mi caso: 4dm1nsql
	shell:> passwd mysql

 7.5º Dejar los privilegios de root:
	shell:> exit

 8.1º Obtener privilegios de usuario mysql:
	shell:> su mysql

 8.2º Ejecutar el script de instalación de la base de datos mediante:
	shell> scripts/mysql_install_db --user=mysql

     Se obtendrá en pantalla:
     "
     Installing all prepared tables
     Fill help tables

     To start mysqld at boot time you have to copy support-files/mysql.server
     to the right place for your system

     PLEASE REMEMBER TO SET A PASSWORD FOR THE MySQL root USER !
     To do so, start the server, then issue the following commands:
     ./bin/mysqladmin -u root password 'new-password'
     ./bin/mysqladmin -u root -h yorerocray password 'new-password'
     See the manual for more instructions.

     You can start the MySQL daemon with:
     cd . ; ./bin/mysqld_safe &

     You can test the MySQL daemon with the benchmarks in the 'sql-bench' directory:
     cd sql-bench ; perl run-all-tests

     Please report any problems with the ./bin/mysqlbug script!

     The latest information about MySQL is available on the web at
     http://www.mysql.com
     Support MySQL by buying support/licenses at http://shop.mysql.com
     "
     -----------------------------------------------------------

	shell:> su mysql

	shell:> cd /data/mysql

	shell:> bin/mysqld_safe --user=mysql &

 ------------------------------------------------------------------------------------
 Para agregar un usuario en la base de datos MySQL
 ------------------------------------------------------------------------------------

 1º Obtener los privilegios del usuario mysql:

	shell> su mysql

 2º Iniciar el servidor de base de datos MySQL mediante usuario mysql:
	
	shell> /data/mysql/bin/mysqld_safe -u mysql --log &

 3º Conectarse al servidor de base de datos MySQl y loguearse 
   mediante el usuario root, especificando que se va usar la base de datos "mysql":
	
	shell> /data/mysql/bin/mysql --user=root mysql -p

 4º Para añadir un nuevo usuario al motor de bases de datos MySQL, en la base de datos
   llamada "mysql" se recomienda según la documentación usar la sentencia GRANT:

 4.1º Para admitir al usuario "user" el acceso a MySQL, con todos los privilegios,
     desde el mismo host donde se ejecuta el servidor de base de datos:
   	
	mysql> GRANT ALL PRIVILEGES ON *.* TO 'user'@'localhost'
	    -> IDENTIFIED BY 'some_password' WITH GRANT OPTION;

 4.2º Para admitir al usuario "user" el acceso a MySQL, con todos los privilegios
     desde cualquier otro host:
     	
	mysql> GRANT ALL PRIVILEGES ON *.* TO 'user'@'%'
	    -> IDENTIFIED BY 'some_password' WITH GRANT OPTION;
